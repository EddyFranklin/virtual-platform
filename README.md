### Installation
```bash
$ composer install
$ npm install # or yarn 
```

### Configuration:
```bash
$ cp .env.example .env
$ php artisan key:generate
```
Then configure the database in .env

### Dependencies:

- [vuejs 2.3.4](https://vuejs.org/)
